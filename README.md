# Cyber Security Internship Projects

Developed by Anas Ahmed

This repository contains Python projects completed during my cybersecurity internship.

## Projects

### Task 01 - Caesar Cipher Tool
Encrypt and decrypt messages using Caesar Cipher with brute force support.

### Task 02 - Image Encryption Tool
Encrypt images using XOR, Reverse, and Pixel Shift methods.

### Task 03 - Password Strength Checker
Checks password strength using entropy, symbols, uppercase, numbers.

### Task 04 - Keyboard Activity Logger
Safe educational keyboard input monitoring tool.
